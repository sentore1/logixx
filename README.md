# Flash Logix - Rwanda Logistics Company

Flash Logix is Rwanda's leading logistics and freight forwarding company, providing comprehensive supply chain solutions across East Africa and beyond.

## Services

- **Freight Forwarding**: Air, sea, and road transport solutions
- **Warehousing**: Secure storage and inventory management
- **Customs Clearance**: Import/export documentation and compliance
- **Supply Chain Management**: End-to-end logistics solutions
- **Real-time Tracking**: Monitor shipments 24/7

## Getting Started

This is a [Next.js](https://nextjs.org) project. To run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
